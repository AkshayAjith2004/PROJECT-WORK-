# Exam Seating Arrangement Website

Frontend website for the College of Engineering Poonjar exam seating arrangement system.

## Features

- Signup and login for `admin`, `teacher`, and `student`
- Teacher-only creation, editing, and deletion of seating schedules
- Student-only viewing of their own allotted seating details
- Department support for `BTech`, `BCA`, and `MCA`
- Clean responsive user interface
- Browser `localStorage` used for demo data and persistence

## Run

Open `index.html` in a browser.
